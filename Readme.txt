# Convertidor TXT a Excel

## Descripción
Aplicación de Streamlit que convierte archivos `.txt` delimitados por `|` en archivos Excel separados por número de factura (`NRO.FACTURA`).

## Requerimientos
- Python 3.7 o superior
- Streamlit
- Pandas
- Openpyxl

## Ejecución Local
1. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```
